# Official Python API Client for the Sms77.io SMS Gateway


## Installation
```shell script
pip3 install sms77api
```


### Example
```python
from sms77api.Sms77api import Sms77api
client = Sms77api("InsertYourSuperSecretApiKeyFromSms77.Io!")
print(client.balance())
```
